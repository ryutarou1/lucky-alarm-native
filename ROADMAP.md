# Lucky Alarm 開発ロードマップ

## コンセプト
「毎朝ランダムにn分早く起きて、得した気分になれる目覚ましアプリ」

---

## Phase 1: コア体験の強化 👈 次回ここから

### 1.1 ジャックポット演出
- [ ] アラーム停止時にコインが降り注ぐアニメーション
- [ ] 振動フィードバック
- [ ] サウンドエフェクト

### 1.2 時間の価値換算
- [ ] 「今日の17分 = コーヒー1杯分」のような換算表示
- [ ] 累計時間を「映画○本分」「本○冊分」に変換
- [ ] カスタマイズ可能な換算カテゴリ

### 1.3 時間の使い道タイマー
- [ ] 「何に使う？」選択画面
  - ☕ コーヒータイム
  - 📱 SNSチェック
  - 🧘 ストレッチ
  - 📖 読書
- [ ] 選択後に自動タイマースタート
- [ ] 二度寝防止効果

### 1.4 二度寝防止（生存確認）
- [ ] アラーム停止10分後に確認通知
- [ ] 確認しないとコインが半減
- [ ] 歩数カウント連動（オプション）
